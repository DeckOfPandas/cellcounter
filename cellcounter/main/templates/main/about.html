About us

CellCountr was conceived and developed during NHS Hackdays – Liverpool 2012 and Oxford 2013, by a team of haematologists, junior doctors, medical students, clinical scientists and professional developers.

We aim to make bone marrow aspirate diagnosis faster, more accurate, and educational.

If you would like to be involved we would love to hear from you: info@cellcountr.com
We would welcome your feedback: feedback@cellcountr.com

Follow us on twitter for updates: Follow @cellcountr

The source code can currently be found here: https://github.com/oghm2/cellcounter

The source code is released under the MIT License (MIT). Details can be found on the Github repository
